This repository contains the projects completed as part of my virtual internship at Horizon Intern.

- Performed data cleaning and preprocessing
- Visualized survival rates based on different features

- Predicted house prices based on features

### 3. Movie Recommendation System

- Used content-based filtering (genres, overview)
- Recommended similar movies using cosine similarity

To apply Data Science and Machine Learning concepts on real-world datasets and build beginner-friendly projects.
